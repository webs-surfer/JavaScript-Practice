# JavaScript Practice 🚀

Welcome to my **JavaScript Practice** repository! This repository documents my JavaScript learning journey through concise notes and hands-on projects. It serves as both a learning resource and a record of my progress as I explore JavaScript concepts and build practical applications.

## 📂 Repository Structure

```text
JavaScript-Practice/
│
├── Notes/        # JavaScript concepts, syntax, and reference notes
├── Projects/     # Mini projects built using JavaScript
└── README.md
```

## 📖 What's Inside

### 📝 Notes

The **Notes** folder contains organized explanations and examples covering JavaScript fundamentals and advanced concepts, including:

* Variables and Data Types
* Operators
* Conditional Statements
* Loops
* Functions
* Arrays and Objects
* Strings
* DOM Manipulation
* Events
* ES6+ Features
* Asynchronous JavaScript
* Fetch API
* Local Storage
* Error Handling
* Object-Oriented Programming (OOP)

### 💻 Projects

The **Projects** folder contains practical JavaScript projects that reinforce the concepts learned through coding. Each project focuses on applying JavaScript to solve real-world problems and improve programming skills.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/webs-surfer/JavaScript-Practice.git
```

Navigate to the repository:

```bash
cd JavaScript-Practice
```

To run any project, open its `index.html` file in your browser or use the **Live Server** extension in Visual Studio Code.
